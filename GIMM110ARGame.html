GIMM 110 AR Game "Standoff"

For this game I made two of the mini games, one of them was my individual game and the other one was a prospector mini game based on the code of the shooter. For the prospector mini games almost 100% of the code was made by me, with the only code I didn’t make being the spawner which was cannibalized from David’s individual game. The idea of the Prospector mini game took inspiration from some old mobile games and arcade games where you move the player around and have them collect falling items.

I created the Prospector mini game in a new unity project and set it up originally using only the code from my individual mini game. My goal with this game was to make a simplistic yet entertaining mini game that was simple to make and fun to play. It was simple and quick to get working, and when showing it to David he recommended I use his spawner script from his individual game, which he helped me set up. The big advantage his spawner script had over mine is the fact that it could be used to spawn multiple things from one empty object and set up the percentages allowing for the spawner to work better.

Following the end of Thanksgiving TJ thank to his dad learned more about using Git and GitHub and we then created a new project and worked on adding our individual games to it. The prospector Mini game was easy enough to add, but it was a tad bit more complicated with the shooter due to it having more settings that needed to be enabled and the individual prefabs. After I got my two mini games finished and added I started to help where I could, I worked on some game art where I could and set up the main menu. Following the completion of the intro animation I got into a discord VC with TJ and worked on adding sound effects to it, and everyone else in the group think I did a good job with it.

Overall this project was at time quite hectic and chaotic, but overall I think that we managed to make something interesting and unique. This entire project was a huge learning experience, and we all learned something valuable for future group projects and endeavors in and out of GIMM.
